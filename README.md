## Hero Squad
A Spark app that allows you to recruit a well-balanced team of superheroes to fight for a given cause.

#Technologies and framework used
1.java 11

2 .spark core 2.12

3 .gradle

4 .jUnit

# usage
1. Clone this repo
2. Compile  `gradle compileJava`
